# BERT FFN V21 for Alveo U250

This directory contains only the production FP32 V21 P16 full-DSP FFN path.

## Layout

```text
src/                 V21 HLS source and input feeder
scripts/             HLS, report collection, and Vivado report scripts
config/              U250 connectivity and SLR placement
reports/baseline/    accepted V21 P16 full-DSP csynth reports
reports/current/     reports generated by the latest csynth run
reports/postroute/   Vivado implementation reports
build/               disposable HLS, XO, and XCLBIN output
```

Production kernels:

```text
bert_ffn_up_gelu_v21_dotpipe_kernel   -> SLR2
bert_ffn_down_v21_dotpipe_kernel      -> SLR3
```

The FFN-only link also uses `bert_ffn_input_v15_feeder_kernel` on SLR2. The
full model replaces this feeder with the attention output stream.

## Accepted baseline at 300 MHz

| Kernel | Latency cycles | DSP | BRAM | URAM | LUT | FF |
|---|---:|---:|---:|---:|---:|---:|
| UP/GELU | 1,534,088 | 1,120 | 302 | 32 | 90,257 | 178,217 |
| DOWN | 1,646,044 | 1,024 | 76 | 32 | 92,339 | 202,321 |

## Run csynth

Run commands from this directory:

```bash
export HLS_PART=xcu250-figd2104-2L-e
export HLS_CLOCK_NS=3.333
vitis_hls -f scripts/run_vitis_ffn_kernel.tcl
python3 scripts/collect_ffn_kernel_reports.py
```

HLS projects are created under `build/hls/`. The two reports used for review
are copied to:

```text
reports/current/csynth_up_v21.rpt
reports/current/csynth_down_v21.rpt
```

## Build XO

```bash
mkdir -p build/xo build/xclbin

v++ -c -t hw --platform "$PLATFORM" --save-temps \
  --hls.clock 300000000:bert_ffn_input_v15_feeder_kernel \
  -k bert_ffn_input_v15_feeder_kernel \
  -o build/xo/bert_ffn_input_v15.xo \
  src/bert_ffn_input_feeder_kernel.cpp

v++ -c -t hw --platform "$PLATFORM" --save-temps \
  --hls.clock 300000000:bert_ffn_up_gelu_v21_dotpipe_kernel \
  -k bert_ffn_up_gelu_v21_dotpipe_kernel \
  -o build/xo/bert_ffn_up_v21.xo \
  src/bert_ffn_kernel_v21.cpp

v++ -c -t hw --platform "$PLATFORM" --save-temps \
  --hls.clock 300000000:bert_ffn_down_v21_dotpipe_kernel \
  -k bert_ffn_down_v21_dotpipe_kernel \
  -o build/xo/bert_ffn_down_v21.xo \
  src/bert_ffn_kernel_v21.cpp
```

## Link U250 bitstream

```bash
v++ -l -t hw --platform "$PLATFORM" --save-temps \
  --config config/ffn_kernel_u250_v21.cfg \
  -o build/xclbin/bert_ffn_v21_u250.xclbin \
  build/xo/bert_ffn_input_v15.xo \
  build/xo/bert_ffn_up_v21.xo \
  build/xo/bert_ffn_down_v21.xo
```

`build/` is generated output and can be deleted before a clean rebuild.
